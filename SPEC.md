# 金山办公套件 (KingOffice) - 技术规格说明书

## 1. 项目概述

**项目名称**: 金山办公套件 (KingOffice)  
**项目类型**: 桌面办公软件  
**核心功能**: 兼容WPS和Office的全功能办公套件，支持文档、表格、演示文稿的创建和编辑  
**目标用户**: 需要轻量级、高兼容性办公软件的个人和企业用户

## 2. 技术架构

### 2.1 技术栈
- **框架**: Tauri 2.x (Rust后端 + Web前端)
- **前端**: React 18 + TypeScript + Vite
- **UI组件**: 自定义组件库
- **文件解析**: 
  - docx: mammoth.js
  - xlsx/xls: xlsx.js
  - csv: papaparse
  - pptx: pptxgenjs
  - pdf: pdfjs-dist
- **加密**: AES-256-GCM (Rust后端)
- **日志**: tracing crate (Rust)

### 2.2 目标平台
- Windows 10/11 (x64)
- 单exe绿色打包
- 无需安装运行库
- 无需联网使用

## 3. 功能规格

### 3.1 核心功能模块

#### 3.1.1 登录与账户系统
- **首次启动引导**: 创建账户界面
- **账户数据**: 用户名、密码（加密存储）
- **加密方式**: AES-256-GCM
- **存储位置**: 安装目录/config/account.enc
- **登录验证**: 本地验证，无需联网

#### 3.1.2 文件管理器
- **支持格式**:
  - 文档: .doc, .docx, .wps, .txt, .rtf
  - 表格: .xls, .xlsx, .csv, .et
  - 演示: .ppt, .pptx, .dps
  - PDF: .pdf
- **文件操作**: 新建、打开、保存、另存为
- **最近文件**: 记录最近打开的文件

#### 3.1.3 文档编辑器
- 富文本编辑
- 字体样式（粗体、斜体、下划线）
- 段落格式（对齐、行间距）
- 插入图片、表格
- 导出为docx/txt/pdf

#### 3.1.4 表格编辑器
- 单元格编辑
- 公式支持（基础）
- 格式刷
- 图表支持
- 导出为xlsx/csv

#### 3.1.5 演示编辑器
- 幻灯片管理
- 模板支持
- 动画效果（基础）
- 导出为pptx/pdf

#### 3.1.6 PDF查看器
- PDF渲染显示
- 缩放、翻页
- 打印支持

### 3.2 进程管理
- **子进程追踪**: 使用Rust的process模块追踪所有子进程
- **退出清理**: 程序退出时自动终止所有子进程
- **信号处理**: 捕获SIGINT/SIGTERM信号
- **守护机制**: 确保进程树完全关闭

### 3.3 日志系统
- **日志框架**: tracing crate
- **日志级别**: DEBUG, INFO, WARN, ERROR
- **日志格式**: [时间戳] [级别] [模块] 消息
- **日志文件**: 安装目录/logs/kingoffice-{date}.log
- **日志轮转**: 按日期分割

## 4. UI/UX规格

### 4.1 窗口结构
- **主窗口**: 1200x800 (最小800x600)
- **模态对话框**: 登录、设置、关于
- **标题栏**: 自定义标题栏，包含程序图标、最小化、最大化、关闭

### 4.2 视觉设计
- **主色调**: #1E3A5F (深蓝)
- **强调色**: #E6B800 (金色)
- **背景色**: #F5F6F8
- **文字色**: #333333
- **图标风格**: 财神主题（用户提供图标）
- **字体**: 微软雅黑 / Microsoft YaHei

### 4.3 组件列表
- 标题栏按钮 (最小化、最大化、关闭)
- 菜单栏
- 工具栏
- 文件树
- 编辑区
- 状态栏

│   └── account.enc   # 加密账户文件

## 6. 安全规格

### 6.1 账户安全
- 密码使用Argon2id哈希
- 账户数据AES-256-GCM加密
- 无明文存储

### 6.2 数据安全
- 文件操作写入日志
- 异常操作记录

## 7. 性能目标

- 启动时间: < 2秒
- 内存占用: < 150MB (空闲状态)
- 文件打开: < 1秒 (10MB以内文件)
- 兼容低配电脑 (2GB RAM可用)
